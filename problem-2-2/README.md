# 자료구조와 알고리즘 2주차 과제 - 괄호의 짝이 맞는지 확인하기

문자열에 포함된 괄호 짝이 맞는지 확인하는 함수를 구현해 주세요.
괄호의 짝이 맞다는 것은 (,[,{ 문자로 열렸을면 반드시 ),],}로 닫혀야 합니다.
- 예를 들어 [()]{}{[()()]}는 각 괄호들의 짝이 맞기 때문에 true를 출력하고
)(({}[])는 짝이 맞지 않으므로 false를 출력해야 합니다.

## 요구 사항

구현한 스택 자료구조를 활용하여 문제를 해결해 주세요

### 입력 예시

| string | return |
| --- | --- |
| '{([])}' | true|
| '[(])' | true |
| ')[](' | false |
| ')[{}()[]]' | false |
| '([{}[]]{)})' | false |
