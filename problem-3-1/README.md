1. 큐의 정의에 따라서 API를 설계해 주세요.

| 함수 시그니처 | 설명 |
| ----------- | ----------- |
| Queue() | 큐를 생성합니다. |
| enqueue(item: any): void | 큐에 아이템을 삽입합니다. |
| dequeue(): item | 큐에서 값을 출력합니다. |
| isEmpty(): boolean | 큐에 값이 비어있으면 true, 아니면 false를 반환합니다. |

